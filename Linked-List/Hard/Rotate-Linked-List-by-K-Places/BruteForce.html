Welcome! We'll rotate this linked list RIGHT by 2 positions using the brute force method. This means we'll move the last node to the front, and repeat this process 2 times. Click "Next Step" to begin!

üìú Brute Force Algorithm:

function rotateRight(head, k) { for (i = 0; i < k; i++) { // Find second-to-last node prev = null; curr = head; while (curr.next != null) { prev = curr; curr = curr.next; } // Detach last node prev.next = null; // Move last to front curr.next = head; head = curr; } return head; }
