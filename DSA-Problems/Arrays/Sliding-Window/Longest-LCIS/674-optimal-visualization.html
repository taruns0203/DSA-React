Longest Continuous Increasing Subsequence

Optimal: Single Pass (Sliding Window)
Time: O(N) | Space: O(1)
⚙️ Controls
Sliding Window
ℹ️ Logic
1. Iterate i from 0 to N-1.
2. If nums[i] > nums[i-1], currentLen++.
3. Else, reset currentLen = 1.
4. Update Max Length at each step.
