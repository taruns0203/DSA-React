Longest Mountain in Array

Improved: Two Pass (Precomputation)
Time: O(N) | Space: O(N)
⚙️ Controls

ℹ️ Logic
1. Pass 1 (Left→Right): Fill up[i]. If arr[i] > arr[i-1], up[i] = up[i-1] + 1.
2. Pass 2 (Right→Left): Fill down[i]. If arr[i] > arr[i+1], down[i] = down[i+1] + 1.
3. Scan: Maximize up[i] + down[i] + 1 where up, down > 0.
