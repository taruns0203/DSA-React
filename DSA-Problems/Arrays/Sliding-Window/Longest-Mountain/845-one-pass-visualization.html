Longest Mountain in Array

Optimal: One Pass (State Machine)
Time: O(N) | Space: O(1)
⚙️ Controls
One Pass Scan
ℹ️ Logic
Iterate i from 1 to N. Track up and down lengths.
- Reset if valley or flat.
- Logic: up++ if climbing, down++ if falling.
- If up > 0 and down > 0, update maxLen.
