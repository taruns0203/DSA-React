🌀 Wiggle Sort II

Optimal - Virtual Indexing (3-Way Partition)
Time: O(N) | Space: O(1)
⚙️ Controls
🎯 Array (Physical Layout)
Virtual Map A(i) => nums[(1 + 2*i) % (n | 1)]
🤔 3-Way Partition Logic (on Virtual Index)
Val > Median: Swap(A(i), A(left)), left++, i++ (Put at Odd idx)
Val < Median: Swap(A(i), A(right)), right-- (Put at Even idx)
Val == Median: i++ (Leave in middle)
📋 State
Median:1
i (Virtual):0
Map(i) (Physical):1
Left (Start):0
Right (End):N-1
