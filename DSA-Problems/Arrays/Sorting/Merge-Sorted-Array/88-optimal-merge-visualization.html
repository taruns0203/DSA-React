Merge Sorted Array

Optimal: Two Pointers (From End)
Time: O(M+N) | Space: O(1)
⚙️ Controls
arrays
nums1 (Size: m + n)
nums2 (Size: n)
🤔 Logic
If nums1[p1] > nums2[p2]: Place nums1[p1] at p, p1--
Else (nums2 larger/equal): Place nums2[p2] at p, p2--
Decrement p
📊 Pointers
p1 (nums1 end):-
p2 (nums2 end):-
p (Total end):-
📝 Execution Log
